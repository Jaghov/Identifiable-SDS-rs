# M4 milestone — Training CLI, optimizers, checkpoints

**PRD:** [§9 Milestones](PRD-burn-port.md#9-milestones) (row **M4**), [§5.2 Model / training](PRD-burn-port.md#52-model--training), [§7 Technical design](PRD-burn-port.md#7-technical-design-high-level).

**Status:** merged — **`snlds-train`** crate + CLI on **`Autodiff<NdArray<f32>>`** (CPU); see **Updates** for PRD gaps (LR schedule, M-Viz+ wiring).

---

## Goal

Deliver a **user-facing training path** on top of **[M3](M3.md)** model core: load **[M1](M1.md)** tensors, run **minibatch** training with **Adam** (Burn `optim`), **gradient clipping**, and **checkpoint** save/load for resume. **Factored** encoder from M3; recurrent encoder remains stretch.

Logged metrics to stdout: mean negated ELBO and reconstruction term per epoch (Rerun via **[M-Viz+](M-Viz+.md)** is optional follow-up).

---

## Implementation summary (merged)

| Piece | Location / notes |
|-------|------------------|
| Crate | **`crates/snlds-train`** |
| Binary | **`snlds-train`** (`cargo run -p snlds-train --bin snlds-train -- …`) |
| Data | Reads **`metadata.json`** + **`sequences.safetensors`** → tensor **`obs_train`** `[N, T, D]` via [`snlds_train::load_train_obs`](../crates/snlds-train/src/data.rs) |
| Model config | [`SnldsConfig`](../crates/snlds-model/src/model.rs) from manifest (`dim_obs`, `dim_latent`, **`hidden_dim`** from CLI, `num_states`) |
| Optimiser | **`AdamConfig`** + **`GradientClippingConfig::Value(grad_clip)`** → **`GradientsParams::from_grads`** → **`optimizer.step(lr, model, grads)`** |
| Backend | **`burn` features `ndarray`, `autodiff`** — `Autodiff<NdArray<f32>>` for practical CPU training speed |
| Checkpoints | **`CompactRecorder`** → **`checkpoint_{epoch:04}.mpk`**; **`resume`** loads record then continues training |
| Tests | `train_one_step_no_panic`, `checkpoint_round_trip`, **`snlds-train --help`** |

## Example (after M1 data generation)

```sh
cargo run -p snlds-data --bin snlds-gen -- --seed 42 --dim-obs 2 --dim-latent 2 --num-states 3 \
  --seq-length 64 --num-samples 128 --data-type cosine --out ./out/run1

cargo run -p snlds-train --bin snlds-train -- \
  --data-dir ./out/run1 \
  --output-dir ./out/run1/checkpoints \
  --epochs 50 --batch-size 32 --lr 0.0003
```

Resume:

```sh
cargo run -p snlds-train --bin snlds-train -- \
  --data-dir ./out/run1 \
  --output-dir ./out/run1/checkpoints \
  --epochs 10 \
  --resume ./out/run1/checkpoints/checkpoint_0049.mpk
```
