# M3 milestone — VariationalSNLDS core (`factored` encoder first)

**PRD:** [§9 Milestones](PRD-burn-port.md#9-milestones) (row **M3**), [§4 Scope](PRD-burn-port.md#4-scope).

**Status:** planned — checkboxes open until implementation merges.

---

## Goal

Implement **burn `Module` / `Autodiff`**-compatible **VariationalSNLDS**-equivalent model matching Python [`VariationalSNLDS`](../identifiable-SDS/models/VariationalSNLDS.py) and supporting modules in [`modules.py`](../identifiable-SDS/models/modules.py): transition heads per discrete state, decoder, \(\pi\) and \(Q\), Gaussian latents with init/emission structure, **`inference='alpha'`** ELBO path using **`log_Z`**, and training-time tensors for reconstruction / KL / switching terms.

**Encoder policy:** Ship **`factored` (MLP)** encoder first for integration parity and tests; **`recurrent` (bi-LSTM → causal LSTM)** remains **stretch / follow-on** unless promoted into M3 ([PRD §4](PRD-burn-port.md#4-scope) encoder row). Prefer a clean **trait or enum** seam so **`encoder_type`** can grow without rewriting inference.

**Kernel dependency:** Discrete-state HMM messaging (**[M2](M2.md)**) and local-evidence plumbing should be **wired or duplicated minimally** so `_compute_elbo` / `_compute_posteriors` analogue can call the same numerics as Python.

---

## Crate layout (informative)

Align with PRD [§7 Technical design](PRD-burn-port.md#7-technical-design-high-level): new workspace member **`snlds-model`** (or equivalent) for Burn `Module` graphs; keep **`snlds-data`** IO-only; **`snlds-core`** may host shared numerics **or** fold into `snlds-model` — **pick one layout** in implementation and record it under **Updates** below.

## Completion checklist

- [ ] Workspace member **`snlds-model`** (name TBD) with **`Module` structs** for SNLDS + **factored** encoder.
- [ ] **Transition nets** per discrete state \(K\); **decoder** \(p(x\mid z)\); **\(\pi\)** / **\(Q\)** logits with **temperature** matching Python training defaults.
- [ ] **Gaussian** reparameterization path for \(q(z)\); emission / transition cov handling with **eps** regularization pattern mirroring Python (`~1e-6` identity add).
- [ ] **`_compute_local_evidence`**, **`_alpha` / `_beta` / `log_Z`** invoked from **[M2](M2.md)**-parity code or inlined equivalent in autodiff graph.
- [ ] **`_compute_elbo`** analogue (reconstruction factorized Normal, KL terms, MSM / switching term wiring as in Python for **`inference='alpha'`**).
- [ ] **Forward** usable on tiny batches from **[M1](M1.md)** SafeTensors (`latents_*` / `obs_*` / optional `states_*`) after tensor conversion.
- [ ] **Unit tests:** small **\(N,T\)** forward + finite loss; optional parity slice vs exported PyTorch tensors (same tolerance policy as **M2**).
- [ ] [docs/PRD-burn-port.md](PRD-burn-port.md) **§8.5** updated for any new first-party deps.

## Testing requirements

Inherit **M0** gates: `cargo fmt --check`, `cargo clippy --workspace --all-targets -- -D warnings`, `cargo test --workspace`.

| Area | Requirement |
|------|-------------|
| **Forward + loss** | One **CPU** minibatch; **finite** ELBO / loss components; no NaN after reasonable init |
| **Shapes** | Batch / time / dim_latent / dim_obs / **K** match declared config |
| **Autodiff** | At least one **gradient** smoke (small parameter subset) comparable to **M0** autograd smoke |
| **Parity (optional)** | Stricter tests promoted when reference tensors exist |

**Out of scope for M3:** Rerun training dashboards (**[M-Viz+](M-Viz+.md)**); full **training CLI** polish (**[M4](M4.md)**); **NeuralMSM** (**[M5](M5.md)**); **recurrent** encoder unless explicitly moved into this milestone.
