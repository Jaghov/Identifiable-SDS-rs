# M5 milestone — Optional NeuralMSM warm-start

**PRD:** [§9 Milestones](PRD-burn-port.md#9-milestones) (row **M5**), [§5.3 Optional NeuralMSM](PRD-burn-port.md#53-optional-neuralmsm).

**Status:** planned (optional milestone) — checkboxes open until implementation merges.

---

## Goal

Port **optional** **NeuralMSM**-style pretraining and **parameter transfer** into **VariationalSNLDS** so training matches the spirit of Python [`NeuralMSM.py`](../identifiable-SDS/models/NeuralMSM.py) + [`train_snlds.py`](../identifiable-SDS/train_snlds.py) init paths: fit on **PCA-reduced** (or equivalent) observations, then copy **transitions**, **\(\log Q\) / \(\log \pi\)**, **init_mean**, **init_cov** (and related tensors) into the main model before **[M4](M4.md)** training. Gate this behind **CLI / config** flags so default runs do **not** require MSM.

---

## Completion checklist

- [ ] **Config / CLI** surface: `--msm-init` (or project naming) with documented **defaults** vs Python scripts.
- [ ] **PCA** (or parity substitute) on observation windows to **`dim_latent`** before MSM fit (**[nalgebra](PRD-burn-port.md#82-likely-needed-for-parity)** or small SVD implementation — align with PRD §8.2).
- [ ] **NeuralMSM** analogue: forward–backward / EM-style loop consistent with Python reference where feasible; **document** intentional simplifications.
- [ ] **Transfer:** load initialized **\(Q\), \(\pi\)**, **init Gaussian** params into **[M3](M3.md)** model; **unlock schedule** for \(\beta\) / **\(Q\)/\(\pi\)** gradients matches Python **or** deviations documented in **Updates**.
- [ ] **Tests:** MSM fit on **tiny** synthetic data — **finite** likelihood / loss; transfer does **not** panic; optional **parity** checkpoint vs Python export.
- [ ] **Documentation:** README section + PRD [§12](PRD-burn-port.md#12-open-questions) updates if checkpoint / flag story is resolved.
