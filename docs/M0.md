## Testing requirements

Minimum bar (**local + CI**):

| Check        | Rule |
|--------------|------|
| **Formatting** | `cargo fmt --all -- --check` |
| **Clippy**     | `cargo clippy --workspace --all-targets -D warnings` |
| **Unit tests** | ≥1 Burn `#[test]`: tensor on chosen backend, small op, `assert` values finite |
| **Scope**      | `cargo test --workspace`; no `#[ignore]` without a tracked issue |
| **Platform**   | `ubuntu-latest`, **stable** Rust |

Out of scope for M0: coverage gates, Miri, multi-feature matrices unless backends require them, mandatory doc-tests.

PR checklist: replicate CI locally before merging.
